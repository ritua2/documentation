Columbus is a software for modeling chemical reactions.
To use columbus log into a compute node, then load the module via the command

module load columbus/7.0

After loading the module the various programs will be in the path.

A simple test program you can run using the example file contained in this directory. Copy it into your current directory
and run the cooresponding commnd:

    argos.x

This will automatically find the file and read the input. You should find output files in your directory:

     aoints
     aoints2
     argosls

The file argosls has a result file you can read. 

More example files can be found in /apps/Columbus/7.0/EXAMPLES
