tmux is a terminal multiplexer. A number of terminals can be created, accessed, and controlled from a single screen. 
To use it on Arc, log onto Arc:
ssh -X abc123@arc.utsa.edu
Load the module:
$ module load tmux
Launch the program:
$ tmux

The detailed instructions can be found at https://tmuxguide.readthedocs.io/en/latest/tmux/tmux.html
