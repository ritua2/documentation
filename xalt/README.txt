To test the operation of Xalt, download xalt_app-test.sh and save it to your Arc account.

1.  Set the script as executable.
    $ chmod +x xalt_app-test.sh

2.  Run the script.
    $ ./xalt_app-test.sh

If successful, the script will output:
XALT operation test: successful.

If not, the script will output:
Cannot confirm XALT operation.
